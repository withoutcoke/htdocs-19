== Description ==

Popularis Extra gives you access to demo import for free PopularisWP themes, extra features like widgets, shortcodes or additional Elementor widgets.

This plugin requires PopularisWP theme to be installed.
== Supported Themes ==
*   [Popularis](https://wordpress.org/themes/popularis/)
*   [Popularis Hub](https://wordpress.org/themes/popularis-hub/)
*   [Popularis Star](https://wordpress.org/themes/popularis-star/)
*   [Popularis Writer](https://wordpress.org/themes/popularis-writer/)
*   [Popularis Press](https://wordpress.org/themes/popularis-press/)
*   [Popularis Fashion](https://populariswp.com/popularis-fashion/)

== Installation ==

1. Upload `popularis-extra` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
