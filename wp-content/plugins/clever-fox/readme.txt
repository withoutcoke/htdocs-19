== Description ==

The Clever Fox plugin adds sections functionality to the Startkit theme and Others Nayra's Themes.
Clever Fox is a plugin build to enhance the functionality of WordPress Theme made by Nayra Themes.

= The Clever Fox plugin features include: =

- Premade Section with Dummy Data
- One Click Data Installation
- 4 predefined content sections
- Live content editing 
- Font Typography
- Custom Theme Color
- and many other features


= Clever Fox Compatible Themes: =

- [StartKit](https://wordpress.org/themes/startkit/)
- [StartBiz](https://wordpress.org/themes/startbiz/)
- [Arowana](https://wordpress.org/themes/arowana/)
- [Hantus](https://wordpress.org/themes/hantus/)
